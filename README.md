# Project: Blackhole

# 구현목표

### 본 프로젝트는 Solitare Card Game 중 Blackhole Game을 만드는 것이 목표이다. 주어진 카드를 모두 Blackhole에 연결하면 승리, 더 이상 BlackHole에 카드를 연결하지 못하면 패배한다.
